TFT_eSPI is an Arduino and PlatformIO compatible graphics library built for 32-bit microcontrollers. Optimized for ESP32, ESP8266, Raspberry Pi Pico (RP2040), and STM32, it delivers rendering performance 3–10× faster than generic Arduino libraries — with built-in support for SPI and parallel interfaces, DMA transfers, smooth anti-aliased fonts, touch input, and off-screen sprites.

## Supported Platforms

TFT_eSPI provides processor-specific optimizations for every supported MCU family.

| Processor    | 4-wire SPI | 8-bit Parallel | 16-bit Parallel | DMA Support    |
|--------------|:----------:|:--------------:|:---------------:|----------------|
| RP2040       | ✅         | ✅             | ✅              | Yes (all)      |
| ESP32        | ✅         | ✅             | ❌              | Yes (SPI only) |
| ESP32-S3     | ✅         | ✅             | ❌              | Yes (SPI only) |
| ESP32-C3/S2  | ✅         | ❌             | ❌              | No             |
| ESP8266      | ✅         | ❌             | ❌              | No             |
| STM32Fxxx    | ✅         | ✅             | ❌              | Yes (SPI only) |

## Quick Start

<Steps>
  <Step title="Install the library">
    In the Arduino IDE, open **Sketch → Include Library → Manage Libraries**, search for `TFT_eSPI` by Bodmer, and click **Install**. For PlatformIO, add `bodmer/TFT_eSPI` to your `lib_deps`.
  </Step>
  <Step title="Configure for your hardware">
    Open `User_Setup.h` inside the installed library folder and uncomment the correct driver define (e.g. `ILI9341_DRIVER`) and set your GPIO pin numbers. See the [Configuration guide](/getting-started/configuration) for details.
  </Step>
  <Step title="Upload your first sketch">
    Open any bundled example from **File → Examples → TFT_eSPI**, or follow the [First Sketch guide](/getting-started/first-sketch) to draw text and shapes on your display.
  </Step>
</Steps>

<Tip>
  PlatformIO users can keep display settings inside their project via `build_flags` in `platformio.ini` — this means library upgrades will never overwrite your configuration. See the [Installation guide](/getting-started/installation) for the full setup.
</Tip>
